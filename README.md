```bash

🛡️ Приватность и Безопасность

Проект следует принципам Zero-Trust & Offline-First:

✅ 0 сетевых запросов к внешним API (вакансии работают в демо-режиме)

✅ 0 аналитики и трекинга (Google Analytics, Яндекс.Метрика отсутствуют)

✅ localStorage используется только для сохранения прогресса и настроек

✅ Сертификат хранится локально с возможностью экспорта/импорта

⚠️ Примечание: Валидация сертификата работает на клиенте. Для production рекомендуется перенести проверку подписи на сервер.
